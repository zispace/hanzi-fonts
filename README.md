# 汉字字体收藏

本仓库主要收录常用、优质的汉字字体，资源收集自网络。

> [!IMPORTANT]
>
> 字体版权属于原作者（字体厂商），使用时（特别是商用）请注意版权和相关限制。
>
> 另外，本仓库中文件不一定是最新版本，需要最新版、其他格式、其他字重的文件，可根据需要去官方渠道下载。

- 目录说明：
  - `更多字体`：来自其他字库、个人开发或者开源的字体
  - `补充字体`：补充一些字符数量不大（一万字符以内、缺少常用繁简兼容）但有特色的字体
  - `港台繁体`：补充港台繁体标准的字体文件
- 字体文件统计信息可参见 [字体信息统计](./字体信息统计.tsv)；
- 字体原始来源（部分）可参考 [字体来源资料](./字体来源资料.tsv)。
- 更多字体：
  - 中文字体收集 [zispace/hanzi-fonts-ext](https://github.com/zispace/hanzi-fonts-ext)
  - 常见超大字符集 CJK 字体 [zispace/hanzi-fonts-ext-ultra](https://github.com/zispace/hanzi-fonts-ext-ultra)
    - 相关讨论和统计见 [:link:](https://github.com/CNMan/UnicodeCJK-WuBi06/issues/17)
- 更多中文字体资料可参考
  - <http://www.homeinmists.com/indexComputing.htm>

### 字体许可证

常见的（开源）字体许可协议包括:

- `SIL Open Font License`：字体专门的许可证，这是真正的开源许可证,允许商业使用、修改、分发。
- `Apache License`：软件（代码）许可证，用户可以自由使用、分发和修改字体。
- `MIT License`：软件（代码）许可证，一个非常宽松的协议，用户可以做商业使用而不需要支付版税。
- `Creative Commons License`：著作权（版权）许可证，有多种类型，用户可以共享、修改字体，但需要署名，有些类型不允许商业使用。
